VSC 자주 사용되는 단축키

  1. html 문서 템플릿 작성 : !
  2. 태그 작성 : 태그이름 enter
  3. 동일한 태그 여러개 작성 : 태그명*n
    li*6
  4. 중첩 태그 작성 : 태그명>태그명>태그명>....
    ex. div>ul>ol*3
  5. 인접 태그 : 태그명+태그명+태그명+...
    ex. li+li+li+li+li+li
       div+h1+h2+span
  6. id 속성 선택자 설정 : 태그명#아이디명
    ex. div#name
       div#b
  7. class 속성 선택자 설정 : 태그명.클래스명
    ex. div.layout
       div.widget_text
  8. 태그 및 텍스트 동시 생성 : 태그명{텍스트}
    ex. p{텍스트}
  9. 숫자 삽입(1,2,3,4...) : 태그명.클래스명$ | #아이디명$ *n
    ex. div.form$*3
        div#board$*6
  10. 상위 계층으로 1칸? 이동 (중첩 상황에서 상위 계층으로) : ^태그명
    (^ 의 개수를 설정하여 실행시키면 됨.
    ^^태그명 : 상위 계층 2칸 이동
    ^^^태그명 : 상위 계층 3칸 이동)
    ex. div>ul>li^p
        div>div>p>ol^^p
  11. 복수 선택자 자동완성 기능 : 태그명#아이디명.클래스명1
    ex. div#name.classA
        div#myid.classA.classB
      (여러 class 선택자 가능)
